How to Use This Tool

1

Configure Animation Settings

Set the animation name, duration, delay, iterations, direction, and fill mode. Use advanced settings for more control.

2

Add Keyframes

For each CSS property (transform, opacity, background), add keyframes at different percentages (0%, 25%, 50%, 75%, 100%). Adjust values and easing functions for each keyframe.

3

Preview in Real-Time

Watch your animation play on the preview object. Use playback controls to play, pause, stop, and adjust speed. Try different preview objects (box, circle, star, text).

4

Use Presets or Start from Scratch

Quickly apply pre-built animations or create custom ones. Modify presets to suit your needs.

5

Copy and Implement

Copy the generated CSS code from the appropriate tab (Keyframes, Animation CSS, or Complete Example) and add it to your project.
