How to Use AuraText

1

Customize Text

Enter your text, adjust font size, and choose text and background colors using the controls.

2

Add Shadow Layers

Click "Add Layer" to create multiple shadow layers. Adjust horizontal/vertical offset, blur, and color for each layer.

3

Try Presets

Click on any effect from the Available Aura Effects to quickly apply beautiful shadow combinations.

4

Export & Share

Download as PNG, save as PDF, or copy the CSS code to use in your projects.
