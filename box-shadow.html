How to Use This Tool

1

Add Shadow Layers

Click "Add Layer" to create multiple shadow layers. Each layer can be individually customized.

2

Adjust Properties

Use the sliders and inputs to control:

  • Offset X/Y: Horizontal and vertical shadow position
  • Blur: How blurry the shadow appears
  • Spread: Shadow size expansion
  • Color: Shadow color with opacity control
  • Inset: Toggle between inner and outer shadow

3

Use Presets

Try our pre-designed shadow presets for quick, beautiful effects.

4

Copy & Use

Copy the generated CSS code and paste it into your project. The code updates in real-time as you make changes.
