# AuraText - Glowing Text Effects Generator

A free text effects generator website that allows users to create stunning glowing text effects for logos, designs, posters, and websites.

## Features

- **15+ Glowing Text Effects**: Aura Glow, Purple Aura, Neon Cyan, Bungee 3D, Glitch, Gradient, Outline, Vintage, Comic Book, Elegant, Fire, Ice, Long Shadow, Gold Foil, Chrome
- **Real-time Preview**: See changes instantly as you type
- **Customization**: Adjust colors, font sizes, and apply effects
- **Multiple Export Options**: Download as PNG or save as PDF
- **Copy CSS**: Get the CSS code for your text effect
- **Visitor Counter**: Real-time visitor tracking
- **Responsive Design**: Works on desktop, tablet, and mobile
- **SEO Optimized**: Proper meta tags and structured data
- **Contact Integration**: Direct contact information
- **Related Tools Section**: Links to other useful tools
