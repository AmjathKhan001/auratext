How to Use This Tool

1

Pick Colors from Canvas

Click anywhere on the color canvas to select a color. Use the magnifier (hover over canvas) for pixel-perfect selection. Zoom in/out for better precision.

2

Use Screen Color Picker

Click "Pick from Screen" to use your browser's EyeDropper API. Pick colors from any application, website, or image on your screen.

3

Adjust with Sliders

Fine-tune colors using Hue, Saturation, Lightness, and Alpha sliders. Watch the color update in real-time across all formats.

4

View All Color Formats

Switch between HEX, RGB, HSL, HSV, CMYK, RGBA, and HSLA formats. Click the copy button next to any value to copy to clipboard.

5

Use Quick Tools

Generate complementary colors, create lighter/darker variants, adjust saturation, or get random colors with one click.
