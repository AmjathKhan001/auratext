    Flexbox Cheat Sheet

    Quick reference for CSS Flexbox properties

    Container Properties

    display: flex Creates a flex container flex-direction row | row-reverse | column | column-reverse justify-content Aligns items on main axis align-items Aligns items on cross axis flex-wrap nowrap | wrap | wrap-reverse

    Item Properties

    order Controls display order (integer) flex-grow Ability to grow (number) flex-shrink Ability to shrink (number) flex-basis Initial size before growing/shrinking align-self Overrides align-items for single item

    Common Patterns

    Centering justify-content: center; align-items: center; Equal Columns flex: 1 on all items Navigation Bar flex-direction: row; justify-content: space-between; Card Grid flex-wrap: wrap; gap: 20px; Sticky Footer flex-direction: column; min-height: 100vh;
